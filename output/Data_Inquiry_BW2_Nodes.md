### 1. B1층 (Row 35-38) — 좌표 불일치

Node 666과 679는 Nodes.csv에 존재하지만, 좌표가 지상층에 해당합니다:

- Node 666: x=3,100 / y=15,000 / **z=+16,950mm**
- Node 679: x=3,100 / y=0 / **z=+21,100mm**

BW2의 B1층은 z가 음수(지하)여야 하나, 해당 노드들은 지상 5~6층 높이의 좌표를 가지고 있습니다.

### 2. B2층 (Row 39-42) — Node 476 누락

- Node 476: Nodes.csv 및 MGT 파일에 **존재하지 않음** (삭제된 노드로 추정)
- Node 489: 존재하지만 z=+8,650mm (지상층 좌표)

### 3. B3층 (Row 43-46) — Node 296 누락

- Node 296: Nodes.csv 및 MGT 파일에 **존재하지 않음** (삭제된 노드로 추정)
- Node 277: 존재하지만 z=-4,150mm (B1층 좌표로 B3층과 불일치)
